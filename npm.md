# NPM

## Updates

### General
`npm install -g npm@<version_number>`

### Package
`npm i [-g] <package_name>@<version_number>`

## Package

### Installa
`npm i <package_name>@<version_number>` (optionally `@latest`; `-D` or `--save-dev` as **dev** dependency)

### Uninstall

`npm uninstall <package_name>`
